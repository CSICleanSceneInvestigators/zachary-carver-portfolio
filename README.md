# Zachary Carver Technical Portfolio

A responsive static portfolio for Zachary Carver, Software Systems Engineer, AI Automation Developer, and Technical Systems Lead in Dallas-Fort Worth, Texas.

## Portfolio Features

- Dark technical visual system
- Terminal-inspired hero
- Responsive and accessible semantic HTML
- Technical ownership and stack views
- Filterable project case studies
- Architecture detail dialogs
- Command palette with `Ctrl/Command + K`
- Multiple accent themes
- Local project brief generator
- Vercel-ready static configuration

## Technology

- HTML
- CSS
- Vanilla JavaScript
- SVG assets
- Vercel static hosting

No build command or environment variables are required for the public static site.

Recommended Vercel settings:

- Project name: `zachary-carver-portfolio`
- Framework preset: Other
- Root directory: repository root
- Build command: leave blank
- Output directory: leave blank
